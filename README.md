# MyWebsite UI Showcase Lab

React-based UI playground containing multiple page patterns:

- Landing UI
- Admin dashboard
- E-commerce flow
- Auth screens
- Multi-step form with validation
- Data table with sorting/pagination/actions
- Modal, drawer, and popover examples
- Dark/light mode switching

## Run

```bash
npm install
npm run dev
```

> If npm registry access is restricted, you can still preview a static build shell by serving the repo root.
